**Both options shown in mockups!**

### Currency Auto-Switching:
```javascript
const currencyMap = {
  en: 'INR',  // English → Indian Rupee
  hi: 'INR',  // Hindi → Indian Rupee
  ar: 'AED'   // Arabic → UAE Dirham
};
```

### RTL Detection:
```javascript
if (language === 'ar') {
  document.dir = 'rtl';  // Right-to-left
} else {
  document.dir = 'ltr';  // Left-to-right
}
```
